# PackageAPI
해당 프로젝트는 물류 업무 중 입출고 되는 패키지(package)에 대한 CRUD API 서버를 구성한 프로젝트입니다.

### 1. 개발환경
-   java 17
-   IDE : Intellij
-   Framework: Springboot(3.2.7)
-   Database: H2
-   ORM: JPA

### 2.  구현 기능
- 패키지 등록
- 패키지 조회
- 패키지 수정
- 패키지 삭제
